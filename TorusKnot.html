1. Initialization of TorusKnot

const geometry = new THREE.TorusKnotGeometry(1, 0.2, 160, 16, 3, 2);
const material = new THREE.MeshPhongMaterial({ color: 0xff0000, shininess: 90.0, });
const torus = new THREE.Mesh(geometry, material);

TorusKnotGeometry(radius : Float, tube : Float, tubularSegments : Integer, radialSegments : Integer, p : Integer, q : Integer)

  • radius : torus의 반지름을 나타낸다. 디폴트 값은 1이다.
  • tube : 튜브의 반지름을 나타낸다. 디폴트 값은 0.4이다.
  • tubularSegments : 튜브의 모양을 결정하는 세그먼트의 개수이다. 값이 3이면 삼각튜브, 4면 사각튜브처럼 렌더링된다. 디폴트 값은 6이다. 값이 커질수록 원형 튜브에 가까워진다.
  • radialSegments : 튜브의 옆구리를 이루는 세그먼트의 개수이다. 디폴트 값은 8이다.
  • p : 회전 대칭축으로 휘는 횟수. 디폴트 값은 2이다.
  • q : torus 내부의 원 안으로 휘는 횟수. 디폴트 값은 3이다.

2. About p and q

  • p와 q는 서로소여야 한다. 서로소가 아니라면 약분한 결과로 렌더링된다.
